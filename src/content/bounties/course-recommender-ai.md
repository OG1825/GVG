## Overview

Build an intelligent system that recommends courses to students based on their academic history, interests, and career goals. This tool will help students make better course selections and improve progression through degree programmes.

## Technical Scope

## Features Required

- Recommendation engine using collaborative filtering
- Natural language processing for course description analysis
- Integration with university course database API
- Admin dashboard for monitoring recommendations
- API endpoints for frontend consumption
- Performance monitoring and analytics

## Dataset

- 5+ years of student enrollment data (anonymised)
- 500+ course descriptions and metadata
- Career outcome tracking for graduates

## Technology Stack

- Python (scikit-learn, TensorFlow)
- PostgreSQL for data storage
- FastAPI or Django for backend
- Docker for deployment

## Acceptance Criteria

- Precision score ≥ 0.75 on test set
- <100ms response time per recommendation
- Handle 1000 concurrent API requests
- Full documentation and code comments
- Test coverage ≥ 80%

## Milestone Schedule

- Week 1-2: Data exploration and EDA
- Week 3-4: Model development and training
- Week 5-6: API development and testing
- Week 7: Performance optimisation and documentation
