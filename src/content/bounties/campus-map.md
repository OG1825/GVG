## Overview

Create an interactive, user-friendly campus map web application. Students and visitors should be able to search for buildings, get directions, find facilities (toilets, cafés, libraries), and save favourite locations.

## Must-Have Features

- Interactive map with building labels
- Full-text search for locations and facilities
- Directions with turn-by-turn navigation
- Favourite locations / bookmarking system
- Mobile-responsive design
- Accessibility features (screen reader support)

## Optional Enhancements

- Event location display
- Parking information
- Campus services directory
- Dark mode support
- Offline map caching

## Deliverables

1. Fully functional web application (React recommended)
2. Responsive design (mobile, tablet, desktop)
3. User documentation
4. Deployment to university servers
5. Training materials for campus tour guides

## Technology

- Frontend: React (or similar)
- Mapping: Mapbox GL or Leaflet
- Hosting: University servers / AWS

## Timeline

- Week 1: Setup and map integration
- Week 2-3: Core feature development
- Week 4: Testing and optimization
- Week 5: Deployment and documentation

## Quality Standards

- Lighthouse scores: ≥90 (performance, accessibility)
- Mobile-first responsive design
- Zero console errors
- ADA/WCAG 2.1 AA compliant
